# Word Guessing Game

## About the Project

I created this word guessing game using Python. The player chooses a category and then tries to guess a random 5-letter word selected from that category.

The categories in the game are:

* Fruits
* Foods
* Animals
* Objects
* Cities
* Countries

After each guess, the game provides feedback for every letter:

* Correct and in the correct position
* Correct but in the wrong position
* Not in the word

The game keeps track of the number of attempts needed to guess the word correctly. After winning, the player can choose to play again or exit the game.

## Features

* Multiple categories
* Random word selection
* Letter-by-letter feedback
* Attempt counter
* Play again option

## How to Play

1. Run the program.
2. Choose a category.
3. Guess the hidden 5-letter word.
4. Use the feedback to help find the correct word.
5. Continue guessing until you find the correct word.
6. Choose whether to play another round.
